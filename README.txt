Aplicativo Jaguar

O seguinte projeto tem como objetivo agregar as informações vindas dos principais Monitoramentos
de Sistemas Fovoltaicos. Tais monitoramentos realizam a aquisição de dados por meio de suas 
respectivas aplicações web.

Neste aplicativo temos agregados os seguintes monitoramentos: 

 • SolarEdge - Adicionado no Quinto Commit | 𝘍𝘶𝘯𝘤𝘪𝘰𝘯𝘢𝘭
 • SolarMan - Adicionado no Oitavo Commit  | 𝘍𝘶𝘯𝘤𝘪𝘰𝘯𝘢𝘭
 • Hoymilles - Adicionado no Oitavo Commit |
 • Canadian - Adicionado no Oitavo Commit  |
 • REFUlog - Adicionado no Oitavo Commit   |
 • AuroraVision - Adicionado no Nono Commit|
 • Growatt - Adicionado no Nono Commit     |
 • EcoSolys - Adicionado no Décimo Commit  |
 • Fronius - Adicionado no Undécimo Commit |
